เส้นทางนี้มันไม่ง่าย.. ชื่อหัวข้อของภารกิจที่2 ก่อนเข้าค่ายจริง

<strong>Introduction</strong>

ปีนี้ปีที่สองแล้วที่สมัครค่ายนี้ ฮ่าๆ  คือลังเลระหว่าง Web Programming กับ Web Marketing  เพราะ Web Programming เป็นสิ่งที่เรียนรู้และทำมา(เรียนมาแนวนี้) แต่มีจุดเปลี่ยนตอนปีสองที่ได้เรียนวิชา PRIN MKTG (Marketing) กับอ.เกต (เกตวดี Marumura) ก็ได้หลงใหลใน marketing อ่านบล็อก อ่านหนังสือ ตามทวิตเรื่อยมาเป็นงานอดิเรก ประกอบกับเป็น Marketeer จำเป็นในทีม Outcube เลยรู้สึกตัวเองเป็นเป็ดตัวยักษ์  ตัดสินเอาไรดี ระหว่างสายที่เรียน กับแนวที่สนใจ แรกๆก็แอบเสียดายจะลง Programming ที่รำ่เรียนมา แต่สุดท้ายก็เปลี่ยนใจ อยากลอง Marketing ดู เลยมากรอกใบสมัครค่ายสาขา Web Marketing ไปวันสุดท้าย เหมือนเขียนได้ไม่เต็มที่ เลยไม่ติดตามระเบียบ

<strong>YWC13 มาเข้าค่ายปีนี้ได้ไง</strong>

เริ่มจากไปวิ่งสวนเบญจกิตติกับสิริกรเพื่อนรัก ละถามไถ่กับเพื่อน เพื่อความแน่ใจว่าไป Web Marketing จริงๆ ดีมั้ย ก็ได้คำตอบว่า "แนวแกดี" 55555 โอเค เป็น Marketing จำเป็นถึงจุดนี้แล้ว ประกอบกับ Web Programming สาย Fronted-End ไม่ใช่แนวจริงๆ ให้ทำไปก็คงไม่มีความสุข เรามันสาย Unix (ฮา) เลยตั้งใจจะลงไปเพื่อเก็บความรู้สายนี้ดีกว่า เพราะจากการทำโปรเจ็กต์มาหลายๆงาน แล้วพบว่า <em>ต่อให้งานดีแค่ไหน แต่ถ้าไม่ใช่ product ที่คนต้องการจริงๆ แก้ปัญหาชีวิตได้ ขาดการ Marketing ที่ดีให้คนได้รู้จัก และคิดแผนการทำรายได้ให้อยู่รอดต่อไปได้  product นั้นก็ไม่รอด</em> ก็รู้สึกว่า ที่ผ่านมาพบตัวอย่างเฟลๆ มาเยอะ ตรงนี้ก็เป็นโอกาสอันดีที่จะได้เรียนรู้ในส่วนนี้กับคนที่ทำด้านนี้จริงๆ จะได้เรียนรู้แล้วเอาไปใช้ เอาดิ เราไม่มีอะไรจะเสีย ไม่ติดปีนี้ก็คงบายจ้า ได้ก็ถือเป็นกำไรไปละกัน อีกหนึ่งประสบการณ์ในชีวิตมหาลัย

<strong>กรอกใบสมัครสิ รออะไร</strong>

นั่นสิ รออะไร จากคราวที่แล้วลังเลมากรอกเอาวันสุดท้าย ปีนี้เลยกรอกก่อน deadline ไปสัปดาห์นึง ก็ใช้เวลากรอกอยู่เหมือนกัน เพราะนั่งหารูปตัวเองที่ไม่ใส่ชุดนิสิตหน้าตรง นี่แหละ ไม่ค่อยจะมีรูปดูดี วันต่อมาไปทำบุญที่วัดเลยยืมมือถือเฮียมาเซลฟี่ (ไม่เกี่ยวกับเนื้อหา) ก็ค่อยๆตอบคำถามไปอย่างใจเย็น และตอบให้เยอะๆหน่อย ดูมีความตั้งใจ (ไม่เหมือนงานเผาปีที่แล้ว ความจริงของปีที่แล้วก็ตอบดีนะ -0-)  อ่าห์... กรอกๆไป~~ รู้สึกคำถามสาขามีข้อเดียวเอง ถามกว้างมากด้วย เว็บที่ปฏิวัติวงการดิจิตอล มันคือเว็บที่เราใช้จนเป็นกิจวัตรวิถีชีวิตเราปะ รู้สึกคำตอบมันใกล้ตัวมากๆ แต่ก็พยามตอบไปเยอะๆ แถมเหตุผลให้หน่อย (ความจริงคนอื่นก็น่าจะคิดเว็บเหมือนๆกันนะ ไม่แน่ใจคิดไปเองเปล่า)

<strong>ติดสัมภาษณ์</strong>

ว้าว ติดสัมภาษณ์ด้วย คุณสิริกรอุตส่าห์โทรมาบอก ดีจุง ติดสัมภาษณ์ครั้งแรก
เท่านั้นยังไม่พอ

<strong>มีการบ้านด้วยนะ</strong>

โอ้ว ให้คิดเว็บที่ปฏิวัติวงการดิจิตอล (อันนี้คิดว่ายาก เพราะถ้าคิดได้จริงมันก็คงรุ่งไปแล้ว และเราไม่น่าใช่คนแรกที่คิดได้ อาจจะมีคู่แข่งตรงนี้อยู่แล้ว ถ้าเราไม่ศึกษาให้ดีพอ/ไม่เจ๋งจริง มีช่องโหว่ก็อาจจะโดนทักจุดอ่อนได้ง่าย) เลยทำอีกข้อนึงดีกว่า น่าจะชัวร์กว่า (ความจริงก็คิดโจทย์นี้ไปเผื่อเล่นๆแหละ แต่เลือกอีกข้อชัวร์กว่า) คือปฏิวัติบางวงการธุรกิจ/อุตสาหกรรม เลยคิด product ชื่อ <a href="bit.ly/farmerhub2015">FarmerHub</a> ขึ้นมา (เป็นไรกับ GitHub ปะ 5555) แก้ปัญหาการเกษตร เพราะคิดว่าเกษตรกรไทยน่าจะได้เข้าถึง information มากกว่านี้  จะได้ได้รับการพัฒนาเท่าเทียมแบบเกษตรกรต่างประเทศ ที่มีการวิจัยพัฒนาที่ดี ประมาณนั้น (มีรายละเอียดอีกมากมาย ขอละไว้ละกัน) ช่วงนั้นก็กำลังสอบๆอยู่ ทำไปเท่าที่คิดว่าพอดีละกัน ไม่มากไปน้อยไป

<strong>สัมภาษณ์จริง</strong>

มาถึง มศว. ก่อนตึกเปิด เลยรอข้างล่าง มีคุยๆกับชาวบ้านเล็กน้อย จนถึงตึกเปิดก็ขึ้นไปอย่างใจเย็น 5555 แล้วก็นั่งรอ คืออยากได้คิวแรกๆ (ในเว็บเราลำดับสอง) แต่สุดท้ายก็สัมภาษณ์ตามลำดับลงทะเบียนก่อนหลัง T T ได้คิวที่หก ก็รอต่อไป คิวที่ผ่านไปอย่างเนิ่นนาน แต่ก็ถือว่าคิวของ Web Marketing ผ่านเร็วกว่าสาขาอื่นอ่านอยู่มากมาย ระหว่างรอก็อ่านหนังสือสอบ ComVision ต่อไป กับแต่งๆ slide เล็กๆน้อยๆ พอถึงคราวเรียกลงไป จึงได้รู้จักเพื่อนร่วมสาขามากขึ้น ปีสี่ด้วยกันสองคนเลย ชื่อ พีช กับ แม็ค  รู้สึกว่าแต่ละคนนี่โหดๆกันทั้งนั้น เราจะติดไหมน้า *-*

<strong>ณ ห้องสัมภาษณ์</strong>

เห็นคนก่อนหน้าออกมา เลยเข้าไปเพื่อไม่ให้เสียเวลารอ พอเข้าไปถึงก็โดนเชิญออกมาทันที (กรรมการยังคุยกันไม่เสร็จ) หน้าแตกเลยครัช *-* ก็นั่งรอสักพัก ก็ถึงคิวเข้าไปจริงๆ ก็สวัสดี แล้วพูดว่า "ขอโทษที่บุกเข้ามาโดยไม่ได้รับอนุญาตค่ะ" (คือมาเล่าให้เพื่อนฟังแล้วรู้สึกฮาประโยคนี้ของตัวเองมาก - แล้วจะมาเผาให้บล็อกของตัวเองทำไม = =") กรรมๆที่สัมภาษณ์เราคือ พี่ป้อม ศิวัตร แห่ง mInteraction กับพี่meSayreya แห่ง lnwshop  ก็พูดๆ แนะนำเรื่องของตัวเองไปเรื่อยๆ จนเขาบอกว่า พอ ด้วยสีหน้าที่เรียบเฉย ให้นำเสนอการบ้าน ก็เปิด Prezi ที่ทำมา พูดไปเรื่อยจนพี่เขาบอกว่าเกินเวลาแล้ว ด้วยสีหน้าที่เรียบเฉย แล้วก็ถึงคำถามว่า ทำไมคิดว่าตัวเองถึงควรได้รับเข้า กับจบไปจะทำอะไร ก็ตอบแบบตาสีตาสาไป คือเราก็ยังไม่มั่นใจในตัวเองขนาดนั้น แต่ก็ตอบแนวที่คิดว่าดูดีที่สุดที่จะสรรหาได้จากชีวิตตัวเอง ก็พูดๆจนพี่เขาบอกว่าหมดเวลาเวลา เลยรู้สึกเหมือนโดนไล่ ฮ่าๆ คือไม่ได้มั่นใจในตัวเองเลย คนอื่นๆก็เก่งๆกันเยอะแยะ เอาเหอะ เราก็ได้ทำที่เราทำได้ไปหมดแล้ว อยู่ที่กรรมการจะตัดสินแล้วล่ะ  ออกมาเลยไปหาข้าวกินแล้วอ่านหนังสือต่อไปที่ มศว. 5555 (แล้วสิริกรคนเดิมก็มาเดินนั่งเป็นเพื่อน)

<strong>ประกาศผลแล้ว</strong>

หลังจากสัมภาษณ์ก็เกิดความไม่แน่ใจ เราจะติดเหรอ -0- ก็เล่าๆให้เพื่อนฟังบ้าง (กรรมการเขาดูเฉยๆกับเรามากนะ แถมออกจะแนวไล่ด้วย) แต่ก็ทำไรไม่ได้ รอไป จนถึงวันประกาศผล สิริกรคนเดิมก็มาแปะรูปในเฟสให้ โอ้ว เราติดแล้ว เย่  ^ ^

<strong>โอนเงิน</strong>

ก็ไม่ได้โอนเงินสักที จนไปต่างจังหวัดนี่แหละ ยืมคอมทั่นอ๋องมาโอนด้วยเน็ตเบาบาง พร้อมกับหาสำเนาบัตรนิสิตนานมากๆ Orz ได้โอนก่อนจะลืม  แล้วพี่วิก็แอดมา (ตอนแรกก็ไม่รับแอด เพราะปกติไม่รับแอดคนแปลกหน้า) จนพี่เขาโทรมาบอกให้รับแอดที เลยรับแอด แล้วเพิ่งสังเกตไม่นานว่ามีกรุ๊ป YWC#13 ด้วยเหรอเนี่ย ตอนกลับถึงกทม. เพิ่งรู้แฮะ รู้สึกบ้านนอก
